

# lpl

英雄联盟(lol)职业联赛命令行

英雄联盟职业联赛，简称“LPL”，是《英雄联盟》中国大陆地区的顶级职业联赛。 英雄联盟职业联赛目前共有14支队伍参与，每年进行春季赛和夏季赛两次联赛，每次联赛分为常规赛、季后赛两部分。

## Quick start
```shell
pip3 install -r requirements.txt
```


## 使用方法
### 查看比赛
```python3 lpl.py -l```或 ```python3 lpl.py --list```

### 查看比赛相关新闻
```python3 lpl.py -n```或```python3 lpl.py --news```

### 查看帮助
```python3 lpl.py --help```
